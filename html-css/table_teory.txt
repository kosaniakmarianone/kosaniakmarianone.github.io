					Таблиці
				
<table></table> - створює таблицю в html документі

<tr></tr> - рядок таблиці

<td></td> - стовпець таблиці(записується у рядку tr)

<th></th> - стовпець-заголовок таблиці(записується у рядку tr. Текст буде жирним і вирівнюватиметься по центру)

				Атрибути для таблиці
rowspan -поєднує рядки
colspan - поєднує стовпці


								CSS
								
background: linear-gradient(to right, red, purple) - адає лінійний градієн(тут перше значення це напрямок градієнту, а наступні значення кольори градієнту. За замовчуванням напрям зверху вниз).
