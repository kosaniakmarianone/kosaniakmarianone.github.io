На цьому уроці ми навчимось створювати форми.

Форми створюються з допомогою парного тегу form,і всі елементи форми задаються всередині цього тегу.
Тепер записуємо три тега input 
Тег input є одним з різнобічних елементів форми і дозволяє створювати різні елементи інтерфейсу і забезпечити взаємодію з користувачем. Головним чином <input> призначений для створення текстових полів, різних кнопок, перемикачів і прапорців.

Якщо ми перевіримо,що у нас вийшло,ми побачимо,що у нас з'явилось три поля,давайте задамо для них тип.
Щоб це зробити потрібно написати type="" і для першого поля це буде тип текст,а для другого submit.
Тип відповідає за те,як буде виглядати наше поле,тобто тип
Text - це текстове поле,а тип
Submit - це кнопка.
Іще існує тип password для тегу input,він приховує введені символи,призначений для паролів.

Переглядаємо

По стандарту на кнопці є текст "відправити" щоб це змінити слід написати наступне value="Кнопка"


Давайте створимо іще одне поле
І кожне поле огорнемо в тег <p> щоб перенести на новий рядок
Як бачимо,у нас є поле для вводу логіна,пароля та кнопка для підтвердження

Для того щоб користувач розумів,за що відповідає конкретне поле в HTML5 є атрибут placeholder - тобто це підказка у вигляді тексту,коли ви нажимаєте на поле,вона пропадає.

Якщо ми нажмемо кнопку відправити,вона спрацює,хоча наші поля пусті.
Щоб такого не було,достатньо написати required -і тепер форма не буде відправляти запит,якщо поля не заповнені.

Для того щоб при заході на сторінку  курсор користувача одразу потрапляв на поле вводу потрібно прописати autofocus


Якщо ми хочемо очистити нашу форму,для цього потрібно створити іще одну кнопку та задати їй наступний тип 
type="reset"


Окрім текстової підказки всередині форми можна задати текст біля форми

Щоб це зробити потрібно скористатись парним тегом label і всередині нього вже записуємо текст 

Для того щоб при натисненні на кнопку нас переводило на інший документ потрібно записати наступне в тезі form
і в лапках вказуємо назву потрібного документа.

Залишилось розглянути два останніх типа для тега input
checkbox - це прапорці,дозволяють вибрати більше одного варіанта із запропонованих.
Записується наступним чином input type="checkbox" далі name="назва",зверніть увагу,що ім'я повинне бути однаковим для всіх прапорців.

radio - перемикачі.Використовуються тоді,коли потрібно вибрати один з декількох варіантів.
Записується наступним чином,ім'я також повинне бути у всіх тегах однаковим.
