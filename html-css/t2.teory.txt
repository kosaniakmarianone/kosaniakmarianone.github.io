					HTML
					Атрибути
	Атрибути - це властивості які можна записувати всередині тегів(першого відкриваючого). Атрибути можуть приймати різні значення. 
	
	Приклад:
		<h1 align='center'>Заголовок</h1> - align це арибут для вирівнювання (тут приймає значення 'center'. Відповідно вирівнює посередині.)
		
	align -  арибут для вирівнювання. Приймає значення:
	1)left
	2)center
	3)right
	4)justify (по довжині сторінки)
	...

							CSS
CSS - Каскадні таблиці стилів. Це спеціальна мова стилю сторінок, що використовується для опису їх зовнішнього вигляду.	Стилі записуємо у тезі <style></style> який має бути розташований у тезі <head></head>

Правила написання: 
1) Вибираємо селектор(тег який хочемо змінити)
2) У фігурних дужках записуємо стилі
3) Стилі записуємо так - 'атрибут' : 'значення' ;

color: #9014a3; - Колір тексту
font-size: 40px; - Розмір тексту
font-family: Cursive; - Стиль тексту
