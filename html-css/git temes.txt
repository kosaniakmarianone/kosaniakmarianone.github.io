1) git bash - встановити програму

2) WIN + R - Відкриється вікно 'Виконати'

3) У вікні 'виконати' напишіть - cmd . Відкриється вікно консолі
		
		РОЗКАЗАТИ ТРОХИ ПРО КОНСОЛЬ
		Команди в консолі:
		
			cd 'назва папки'- команда для переходу до папки
		
			cd .. - повернутись назад
	
			dir - команда покаже вміст даної папки
	
			cls - чистити вміст консолі
	
		
4)cd Desktop - переходимо на робочий стіл

5) cd 'Назва вашої папки' - переходимо у свою папку

6) git clone 'адреса вашого репозиторію з GITGUB'  - копіюємо репозиторій з github. наприклад - 'https://github.com/MarikKV/TaskManager.git'

7) cd 'Назва створеної папки з GITHUB'

8) git status - 'перевірка статусу git-репозиторію(чи змінено фали)'
	
	//Робимо зміни у файлах 
	//Після змін можна повторно написати git status. У консолі червоним виведе змінені або добавленні папки і файли
	
9) git add .      -  'добавить всі зміни у чергу'

10) git commit -m "пояснення змін" - 'добавить ваше коротке пояснення внесених змін'

11) git push origin - добавить зміни у github (Виплеве вікно github де потрібно ввести свій логін і пароль)


Якщо робити це на заняттях може виникнути проблема з заливанням на github через збереження логіну і паролю-github у вінді попереднього юзера.
Тому на заняттях перез заливанням роботи, потрібно зробити такі кроки
1) У пошуку знайти Панель керування
2) Облікові записи користувачів
3) Диспетчер облікових записів
4) Облікові данні Windows
5) У списку 'загальні облікові данні' знайти git:https//github.com і видалити його.
