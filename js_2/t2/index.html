Request та Responce. Опрацювання данних отриманих від бази данних (.then(), .catch())

1. Створіть окремий файл у папці firebase з конфігурацією вашої бази данних.

// Файл index.js у папці firebase const firebaseConfig = { apiKey: "...", authDomain: "...", projectId: "...", storageBucket: "...", messagingSenderId: "...", appId: "...", measurementId: "...", }; firebase.initializeApp(firebaseConfig); var db = firebase.firestore();

2. Створіть веб-сторінку та підключіть до неї firebase firestore.

<script src="https://www.gstatic.com/firebasejs/6.0.0/firebase-app.js"></script> <script src="https://www.gstatic.com/firebasejs/6.0.0/firebase-firestore.js"></script> <script src="../firebase/index.js"></script>

3. Додайте у базу данних декілька статтей в колекції articles. Статті зберігайте з такими ключами: title, article, author

4. Показуйте на сторінці спінер доки данні не будуть завантаженні з бази данних.

5. Приховайте спінер та відобразіть статті із бази данних коли вони завантажаться.
