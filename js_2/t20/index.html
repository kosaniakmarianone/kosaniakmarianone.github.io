Курсова робота. Створення корзини.

1. Додайте нові компоненти для показу товарів користувачу. Додайте відповідні посилання у меню та налаштуйте для них routing.

Компонентa OrderProducts

html для orderProducts.html (компоненти OrderProducts)

<div class="orderProducts"> <h1 class="text-center">Products</h1> <div class="d-flex flex-wrap justify-content-around mt-5"> <!--нова коммпонента для відображення товарів (order-card)--> </div> </div>

Vue.js для компонента OrderProducts

<script> const OrderProducts = { template: orderProducts.data, methods: { getAllProducts(){ db.collection("products") .get() .then( res => { data.products = []; res.forEach( element => { const product = { ...element.data(), id: element.id }; data.products.push(product) }) this.$forceUpdate(); }) } }, components: { OrderProductCard //птрібно стоврити нову копоненту для відображення товарів }, mounted: function(){ this.getAllProducts(); } } </script>
