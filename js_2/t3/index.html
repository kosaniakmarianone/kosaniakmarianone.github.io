Додавання нових данних у firebase

1. Створіть 2 веб-сторінки та підключіть до них firebase firestore окремим файлом.

2. Сторнки повинні мати форми для додавання нових об'єктів у firbase - user, article

3. Нові обєкти повинні містити ключі як об'єкти з попередніх завдань.

Article: title, author, text.

User: name, lastName, email

<script> // Додавання нового документу у firebase db.collection("users").add({ name: "Taras", lastName: "Milevich", email: "taras@gmail.com" }); // Aбо const user = { name: "Taras", lastName: "Milevich", email: "taras@gmail.com" } db.collection("users").add(user) </script>
